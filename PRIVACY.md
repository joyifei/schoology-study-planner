# Privacy Policy for Schoology Study Planner

Last updated: May 18, 2026

Schoology Study Planner is a Chrome extension for personal/family use. It helps a parent organize Schoology homework, refresh saved course grade pages, track completed assignments, and calculate GPA.

## Data the Extension Uses

The extension can read visible Schoology page content after the user clicks Sync, Grab, or Update All Grades. This may include homework titles, course names, due dates, grade values, and grade page URLs.

The extension also stores user-entered planner data, including done status, estimated homework duration, Today Plan selections, course setup, course credits, GPA include settings, and grade history.

## How Data Is Stored

All planner data is stored locally in the user's Chrome browser using Chrome storage. The user may also choose to export planner data to a local JSON file and later import that file.

## Data Sharing

The extension does not send planner data to any external server. It does not sell data, use data for advertising, or share data with third parties.

## Login Information

The extension does not collect, store, or transmit Schoology usernames, passwords, authentication tokens, or credentials. The user must already be logged in to Schoology in Chrome.

## Remote Code

The extension does not use remote code. All JavaScript, HTML, and CSS are packaged with the extension.

## Contact

For questions about this privacy policy, contact the extension publisher through the Chrome Web Store listing.
